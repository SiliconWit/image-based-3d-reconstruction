Lenses can be changed, but the sensor is always permanent.
Specs for the sensor include:
	- Resolution
	- Sensor Size, diagonal, width and height
	- Make 
	- Model
The Lens Specs include:
	- Aperture
	- Focal Length
	- Image Format => Brings about the vignette effect.
![Image Format Examples](Pasted%20image%2020230724230729.png)
>**Distortion**: Lenses designed for larger formats might exhibit different distortion characteristics when used on smaller sensors, especially if only the center part of the lens is used. This can impact the accuracy of photogrammetric reconstructions.

A telephoto lens is useful for long range photos.
Wde angle lenses come with high image distortion, moreso towards the edges of the image.
