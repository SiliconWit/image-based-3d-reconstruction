#open3d
Pixel is to 2D as Voxels is to 3D.
A point cloud can be down-sampled to voxels. A bunch of points are averaged out to produce voxels.
With a downsampled point cloud, more efficient computations can be performed against the reconstruction.
> Voxel downsampling uses a regular voxel grid to create a uniformly downsampled point cloud from an input point cloud. It is often used as a pre-processing step for many point cloud processing tasks.
